# Supported File Formats

Platformatic supports the following file formats, their file extensions and the ability to include comments:

| Format | Extensions         | Comments Supported |
|--------|--------------------|--------------------|
| JSON   | `.json`            | No                 |
| JSON5  | `.json5`           | Yes                |
| YAML   | `.yml`, `.yaml`    | Yes                |
| TOML   | `.tml`, `.toml`    | Yes                |
